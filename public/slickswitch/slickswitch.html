SlickSwitches v1.0

Created by Michael Golus and his brother Matt

SlickSwitches is a low-footprint and thus environmentally friendly jQuery plugin that allows you to turn any ordinary boring checkbox on your page into a fantastically well-styled switch, and since the switch takes the place of an ordinary checkbox, it degrades without an issue if JavaScript is not enabled. With it's many possibilities, it's simply lovely.

Documentation

1. First, make sure you are using valid DOCTYPE

This is required for SlickSwitch to look and function correctly.


<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN"
"http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

2. Include nessesary JS files

Loading jQuery from CDN Content Delivery Network is recommended.


<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>

<script src="js/jquery.slickswitch.js" type="text/javascript"></script>

3. Add SlickSwitch CSS file

Don't forget to change background image paths if CSS file is not in a folder named "css" within the root directory and switch.png is not in a folder named "images" within the root directory.


<link rel="stylesheet" href="css/slickswitch.css" type="text/css" />

4. Create a checkbox input element with the "switch" class.


<input type="checkbox" class="switch" />

5. Initiate slickswitch( )


<head>

<script type="text/javascript">

$('.switch').slickswitch();

</script>

</head>

You can pass options as key/value object to slickswitch() function or modify them at the top of SlickSwitch JS file

Key Default Value Description
cssPrefix 'ss-' Prefix of classes generated by slickswitch
useAnimation true When false switch is instant
toggled function( ) { } Trigger a function when switch is toggled
toggledOn function( ) { } Trigger a function when switch is switched on
toggledOff function( ) { } Trigger a function when switch is switched off
